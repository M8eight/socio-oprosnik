## Приложение опросник с элементом интерактива в виде визуальной новеллы (еще не до конца закончено)

#### для старта используйте start.bat или через venv запускайте
```
uvicorn main:app --reload --port 8080 --host 0.0.0.0
```
### запуск будет на 0.0.0.0:8080
#### localhost:8080 главная страница
#### localhost:8080/admin админка для изменения диалогов и загрузки изображений
